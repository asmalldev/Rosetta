To compile and run, simply use myatscc or patsopt to compile the source file in
a problem's sub directory and execute the produced binary.

For example: 
```
cd abc_problem
myatscc abc_problem.dats
./abc_problem_dats
```
output:
```
Testing abc_problem.dats ...

	a:		true
	BaRk		true
	TreAt:		true
	squAd:		true
	conFuSE:	true
	BooK:		false
	ComMoN:		false

All tests passed!
```
